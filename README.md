# Create polar area charts 
A webpage to collect some values and output a polar chart using chart.js

## Requirements
index.html
styles.css
script.js
config.js

# To run without a web server file:// include
app.js 

## How to run
Place .js, .css and index.html files in the webroot director directory


## How to run
1. start a web server -- for example 
```python3 -m http.server 8000```
2. This starts a server on port 8000
3. Navigate a browser to http://localhost:8000/
4. Fill in the form
5. Print or save to CSV

That's it.
